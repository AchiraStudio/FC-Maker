# FC-Maker by Achira

A premium Roster Generation Engine for FC 26, featuring smart potential calculations, realistic attribute generation, and modern aesthetics.

## Key Features
- **Smart Potential Engine**: Dynamic potential boosts based on age, market value, and nationality.
- **Realistic Attributes**: Position-weighted attribute generation (e.g., Stamina logic, GK adjustments).
- **Modern UI**: Glassmorphism design with smooth Framer Motion animations.
- **Demographics Engine**: Accurate birthdate to FIFA format conversion and nationality-based appearance.

## Development
```bash
npm install
npm run dev
```
